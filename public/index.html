Project Management REST API

This API allows users to create, retrieve, update, and delete project records stored in a MySQL database.

API Endpoints

  • POST /projects – Create a new project
  • GET /projects – Retrieve all projects
  • GET /projects/:id – Retrieve a project by ID
  • GET /projects/name/:name – Retrieve a project by name
  • PUT /projects/:id – Update a project by ID
  • DELETE /projects/:id – Delete a project by ID
  • DELETE /projects – Delete all projects
